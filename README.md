# Barik

A lightweight macOS menu bar replacement for use with [yabai](https://github.com/koekeishiya/yabai) or [AeroSpace](https://github.com/nikitabobko/AeroSpace).

> **Note:** This is a personal fork of (a personal fork :D) [mocki-toki/barik](https://github.com/mocki-toki/barik) tailored to my needs. Not accepting contributions. No promises of support or maintenance.
